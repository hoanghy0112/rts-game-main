# Character Asset Sources

The runtime peasant scenes currently use repo-native low-poly fallback geometry so the village population works without committing external download archives.

The intended CC0 source packs for production replacement are:

- Quaternius Universal Animation Library 2: https://quaternius.com/packs/universalanimationlibrary2.html
- Quaternius Modular Character Outfits - Fantasy: https://quaternius.com/packs/modularcharacteroutfitsfantasy.html
- Quaternius Universal Base Characters: https://quaternius.com/packs/universalbasecharacters.html

These packs are listed by Quaternius as CC0. Import only the selected `.glb` character, textures, and animation clips; do not commit the full downloaded `.zip` archives.
