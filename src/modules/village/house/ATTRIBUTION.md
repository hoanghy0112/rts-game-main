# Village House Asset Notes

The active village house wrapper scenes in this directory use local models supplied in `_temp_house`. No license or provenance metadata was present alongside those files, so the models are treated as user-supplied assets with license/provenance pending.

Game-facing copied assets:

- `src/assets/models/village/house/cartoon_hut_3d_modal_pack.glb`
- `src/assets/models/village/house/thatched_hut_on_stilts/model.glb`
- `src/assets/models/village/house/thatched_hut_on_stilts/textures/`

The older procedural house scenes remain local placeholders and can still be used for tests or fallback scenes. The `_temp_house` directory remains a staging/source copy outside the game-facing `res://assets` paths.
